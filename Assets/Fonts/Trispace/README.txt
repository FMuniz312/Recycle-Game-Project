Trispace Variable Font
======================

This download contains Trispace as both a variable font and static fonts.

Trispace is a variable font with these axes:
  wdth
  wght

This means all the styles are contained in a single file:
  Trispace-VariableFont_wdth,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
